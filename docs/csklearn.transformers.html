fit(X, y=None)[fuente]¶

Get columns from X, and keep useful variables and drop useless. This transformer is useful when you are not sure if your new datasets have new columns that you don’t need. In that case, automatically new variables will be droped and passthrought error

Parámetros

  • X (pd.DataFrame) – X matrix with column names.

  • y (array, optional) – y matrix (only for sklearn, not needed). Defaults to None.

Devuelve

X with columns filtered
